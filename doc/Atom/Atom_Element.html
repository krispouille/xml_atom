Atom_Element

Description
Description | Descendents | Methods (details)

this class extends a DOMDcoument by proposing in addition, methods to handle atom elements

thus, an Atom_Feed extends an Atom_Element which also extends a DOMDocument i choose DOMDocument (instead of DOMElement), because it's the only way to handle at the same time nodes, and their display
